Plugin a shortcode pro snadné a spolehlivé vkládání widgetů Hlídače státu do textu.

== Description ==

Plugin vloží do vaší stránky vložený rám (iframe), ve kterém se zobrazí vámi požadovaná stránka z Hlídače Státu.

Plugin neodesílá na server Hlídače Státu žádné osobní údaje či údaje z vašeho WordPressu, ale v iframe používá Google Analytics pro analýzu návštěvnosti.

Podmínky užívání a ochrana soukromí: https://www.hlidacstatu.cz/texty/provoznipodminky/

Smluvní podmínky služby Google Analytics: https://www.google.com/analytics/terms/cz.html
